jsr303使用

JSR 是 Java Specification Requests 的缩写，即 Java 规范提案。存在各种各样的 JSR，简单的理解为 JSR 是一种 Java 标准。
JSR 303 就是数据检验的一个标准。

Hibernate Validator是jsr303的一种实现。

使用

统一异常处理

我们上面用BindingResult来处理，但是当有许多的方法上面要校验时，就得在每个方法里重新写一遍错误封装。

我们可以通过springboot的controllerAdvice来统一异常处理

编写一个异常处理类，添加@ControllerAdvice，对于指定包的类抛出的异常，进行统一处理

分组校验

对于不同的场景，如添加和修改，他们所校验的字段是不一样的，例如添加不需要id，而修改则需要id。

这时可以使用分组校验，对于不同的场景使用不同的分组校验规则。
